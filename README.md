# Proiect Atestat Informatică - Serialul "Supernatural"

Proiect realizat pentru examenul de atestat la informatică (clasa a XII-a) - o platformă web dedicată serialului "Supernatural".

## 🔗 Link Proiect
Pagina poate fi accesată online aici: [Supernatural - Website](https://ioanarusu133.github.io/Supernatural/)

## 📋 Structura Website-ului

* **Home (`index.html`):** Pagina de start care introduce utilizatorul în atmosfera serialului.
* **Personaje (`personaje.html`):** O listă centralizată a figurilor emblematice.
    * *Pagini dedicate:* Fiecare personaj (Sam, Dean, Castiel etc.) are o pagină individuală.
* **Monștri (`monstrii.html`):** O enciclopedie a creaturilor supranaturale.
* **Galerie (`galerie.html`):** Secțiune vizuală interactivă cu navigare.
* **Contact (`contact.html`):** Pagina de prezentare a autorului (foto).

## 🚀 Funcționalități și Elemente Interactive

Proiectul include implementări tehnice specifice pentru a demonstra competențele digitale:

* **Image Maps:** Utilizarea hărților de imagini interactive pentru navigarea către paginile personajelor prin click direct pe elemente vizuale.
* **Galerie Foto:** Implementarea unei galerii dinamice care permite schimbarea imaginilor folosind butoane (săgeți).
* **Multimedia:** Integrare audio pentru fundal, oferind o experiență imersivă.
* **Interfață Utilizator:** Butoane personalizate și layout structurat pentru o navigare intuitivă.

## 🛠️ Tehnologii Utilizate

* **HTML5:** Structură semantică, `<map>`, `<area>`, `<audio>`.
* **CSS3:** Stiluri pentru layout, butoane și design adaptiv.
* **GitHub Pages:** Serviciu de hosting pentru publicarea proiectului.
